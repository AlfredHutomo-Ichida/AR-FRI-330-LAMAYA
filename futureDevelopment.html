Future Development

for the future development of this game there are some adjustment in the team roster role to make this game even better in the future. here are the list of positions needed for future development.

Positions needed

  • Game Designer

    • This role is important is to design the flow of the game and the functionality of the game overall. Game designer is need so that the flow of the game will be much smoother than our final product. Game designer will also make the story of the game and make the whole game more appealing to broader audience.

    Their required skill
    • Creative thinking
    • Teamwork
    • Experience with children

  • UI/UX Designer

    • This role will work together with the game designer to create an appropriate design for the web game. They will make a good layout that is very accessible to all users including users with disabilities. They design the website and later work with the front-end developer to create the final website.

    Their required skill
    • Creative thinking
    • Using design tools to make mockups
    • Basic understanding of web development
    • Teamwork

  • Web Front-end Developer

    • This role is needed to develop the main website inteface of the game they will work together with the UI/UX designer to receate their design using code in HTML, CSS and JavaScript. This role need to be an experienced person in web development and understands the theme of the game very well

    Their required skill
    • Creative thinking
    • Expert understanding of front-end web development (the use of frameworks are welcomed)
    • HTML
    • CSS
    • JavaScript
    • Teamwork

  • Web Back-end Developer

    • If we wish to have special server hosted to save files online. This role will work the back-end of the website which is the server. Back-end developer will make the api and pass it to the front-end to be Processed. This will be an important role for the functionality of this game.

    Their required skill
    • Expert in algorithms
    • Expert understanding of back-end web development (the use of frameworks are welcomed)
    • PHP
    • Python
    • Or NodeJs
    • Teamwork
