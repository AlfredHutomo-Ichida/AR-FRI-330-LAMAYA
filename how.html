How

Tools

Main communication software -> Microsoft Teams Version 1.3.00.21759

Microsoft Teams allows communication between teams become more productive and modern with features; like connecting with apps and also Microsoft office apps like word, powerpoint, and excel to work with teams. We will be using Microsoft Teams since it is the formal communication software used in BITs and it is very good communication tool overall. This software is free

Text/Code Editor -> Visual Studio Code Version 1.48.0

Visual Code studio is a very good and feature packed widely Code Editor used by many web developers. We decide to use Visual Studio Code because of the features it has and it is suitable for web developing. There are many recommendations in the Internet that points to VSC as a great Code Editor and reliable. This software is free

Version control software -> Github

Github is a version control software that is used widely to keep track of versions of source code of a software or even data. We need to use github for to control and let all of us contribute to the HTML documents and also for the final website. The version we are using is free

Web Design / Mockup -> Figma, Balsamiq

Figma is a web based software or can also be downloaded to make designs or wireframes. Figma is great tool for designing an user interface for a website. Same with Figma, Balsamiq is a more basic wireframing compared to Figma which can do more. This strengten the use of Figma more for our project because of the powerful function this software can be used to design a website or an app interface.

Graphic editor -> Procreate Version 5.0.8

Procreate is a graphic editor software developed by Savage Interactive first made for the iOS platform. This software is widely used by artist using iPad for making their arts digitally. We choose this software because our design team is used to be using the software. The software cost $9.99 in the apple appstore for iPad

Cloud Web Hosting -> Amazon Web Service

AWS is a service that provides on-demand cloud computing platforms. Their service lets us host our website so that people can access our website using their own device. We are using AWS because it is provided from RMIT for us to use freely. This service usually are paid but RMIT has provided us with $50 dollars credit to use.

Task Tracker -> Trello

Trello is a cloudbased software that makes task management easier for all people and get teams organised with their tasks. We will be using Trello as our Task management software to track each team members. We will be using the free version of Trello because we do not need to additional features of the paid version.
